# Unconformity Informed Field-adaptive Synthetic Seismic Data Generation via Transfer Learning Enhanced CycleGAN for Fault Interpretation

**This is a Pytorch version of  Transfer Learning Enhanced CycleGAN implemented for fault interpretation modified from the repository "https://github.com/NovemberZZ/Label-To-Seismic-Data/tree/main"**

As described in **Unconformity Informed Field-adaptive Synthetic Seismic Data Generation via Transfer Learning Enhanced CycleGAN for Fault Interpretation** by [Yihuai Lou](https://scholar.google.com/citations?hl=zh-CN&user=Fzw-zCMAAAAJ)<sup>1</sup>, 
[Yusheng Wang](https://scholar.google.com/citations?hl=zh-CN&user=RCwget0AAAAJ)<sup>2</sup>, 
[Xinke Zhang](https://scholar.google.com/citations?hl=zh-CN&user=cQlH6o0AAAAJ)<sup>2</sup>, 
[Naihao Liu](https://www.researchgate.net/profile/Naihao-Liu)<sup>3</sup>, and 
[Yunmin Chen](https://scholar.google.com/citations?hl=zh-CN&user=0giqBqMAAAAJ)<sup>1</sup>.

### Dataset
The data for training CycleGAN named "FFEunconformity.npy", "kerry_cut.npy", and "unconf_cut.npy" can be downloaded from https://drive.google.com/drive/folders/1uWOTtE1Fr8K8I2E7PhmSZpj6Rw1d1iH8?usp=drive_link.
The generated dataset named "FT-CG synthetic seismic data for Kerry.npy" and "FT-CG synthetic seismic data for Opunake.npy" can also be downloaded from https://drive.google.com/drive/folders/1uWOTtE1Fr8K8I2E7PhmSZpj6Rw1d1iH8?usp=drive_link.
